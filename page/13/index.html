安装redis

  1. pull最新镜像

  2. 启动容器

    • –restart always 表示启动失败，总是重新尝试
    • -p 表示映射端口
    • -v 表示将/etc/redis映射到本机的/home/centos/spring/redis/config,以及数据data映射到本地
    • –net docker-nat –ip 192.168.0.63 表示为容器指定网卡并赋予ip，此处可以不需要
    • redis 表示是当前镜像名称，没有版本号
    • redis-server /etc/redis/redis.conf 表示以指定配置文件方式启动redis-server
    • –appendonly yes 表示开启持久化

  3. 查看日志，可能会出现没有找到redis.conf的问题

    从官网下载一个标准配置文件,将配置文件放到指定的/home/centos/spring/redis/config下。并修改配置文件:

    • 修改bind为0.0.0.0
    • 修改protected-mode 为 no
    • 修改 daemonize 为 no 关闭守护进程方式启动,改为yes时以外部文件启动方式会失效

  4. 因为配置过restart always,所以配置正确好，自动重启就启动成功了。通过docker inspect container_id查看容器信息。

通过VMWare制作linux启动U盘

2020-04-03
Linux

以下方法并不能做启动盘，但做移动系统还是可以的

制作

本文主要描述的是制作linux自启动u盘，即没有图形页面，直接通过ssh远程操作机器了。因此首先U盘是引导安装或者这个U盘就是一个简易的Linux系统。我们要使用他只需要将u盘插上主板，并将远程ssh将硬盘挂载上即可。

这是第一个方式，第二种就是引导安装，批量自动安装系统。这个实现起来可能会比较复杂，甚至我也没弄好。

对于没有GVA,HDMI等图形输出接口的设备而适用。但首先你确保你的主板是可以直接U盘启动的。理论上是可以的，不过在启动前你需要把硬盘都拔掉。因为没有硬盘设备，只有U盘输入设备，因此应该是直接从U盘启动的。然后再次插上硬盘，顺序应该就没问题了。不过这个U盘不能拔。

材料:

  • 16/32GU盘
  • 安装VMWare 软件的笔记本或家用电脑,我用的时15版本，不过配置变更不是太大
  • linux的ISO镜像文件

按标准流程来，创建一个虚拟机

流程:

  1. 文件
  2. 新建虚拟机
  3. 自定义
  4. 下一步
  5. 单选 稍后安装操作系统
  6. 下一步
  7. 虚拟机名称定义一下，位置定义一下
  8. 设置cpu数量和内存
  9. 设置网络连接方式为桥接
  10. 后面全都默认即可

编辑虚拟机，选CD/DVD添加iso镜像。

选择USB控制器，根据你的U盘类型，选择兼容2.0还是3.0，然后勾选显示所有USB输入设备。

启动虚拟机，然后插入U盘到电脑，选择虚拟机-可移动设备-你的U盘，选择连接。然后在设置分区的时候就找到U盘了。

启动服务器，一步步安装。在安装的时候，要配置分区的话。你就需要将分区设置到你的U盘上了。这个在网上搜索centos硬盘分区都能找到怎么配。简要地说一下centos7安装中，可以自定义分区，点一下哪个帮助自定义创建，就都给你弄好了。一般是一个/boot配置为标准分区，文件系统都可以选ext4的，一个/根设为LVM逻辑分区，一个swap交换分区，文件系统选为swap。

等待安装就行了。

安装完后会让重启，可以选择重启，但是起不来，此时你要关闭虚拟机，在菜单栏上关掉。

然后编辑虚拟机设置，点击添加，添加一个硬盘，默认选SCSI即可，选择 使用物理磁盘，然后会有设备选择就是硬盘，一般都是最后一个，可以点击使用单个分区按钮之后点下一步，可以看到容量。来确定是不是U盘。知道是哪个后选择使用整个磁盘的单选按钮。设置磁盘文件默认就好。

然后点菜单 虚拟机-电源-打开电源时进入固件，在boot栏里更换一下U盘在前，通过+号可以向上移。随后F10保存并退出，就直接启动U盘里的系统了，可以再这里操作了。

等把必要的ssh，静态ip等都配置好就可以关闭虚拟机，拿出来使用了。

Postgresql常用函数

2020-03-24
数据库

日期函数

获取最近一周，一个月，一年的日期函数

  • 近一周
    select now() - interval ‘1 week’;
  • 近一月
    select now() - interval ‘1 month’;
  • 近一年
    select now() - interval ‘1 year’;
  • 下一周
    select now() + interval ‘1 week’;
  • 下一月
    select now() + interval ‘1 month’;
  • 下一年
    select now() + interval ‘1 year’;

    获取指定时间段内的日期列表

    select to_char(generate_series(‘2018-10-01’::date,’2018-10-25’,’1 day’),’yyyy-mm-dd’);

日期截断,获取当前日期的年月日

date_trunc(‘<unit>’,<date>);

  • 截取为当天日期
    select date_trunc(‘day’,now())
  • 截取为当月日期
    select date_trunc(‘month’,now())
  • 截取小时，即当天0点整合12点整
    select date_trunc(‘h’,now())
    select date_trunc(‘h’,to_date(‘2020-02-12 09:00:00’,’YYYY-MM-DD HH24:mi:ss’))
  • 截取分钟，即当前当前分钟整，秒数归零
    select date_trunc(‘minute’,now())
  • 截取当前季度，以1234位第一季度，当前月为2月则季度获取为1月1日
    select date_trunc(‘quarter’,now())

第二个参数为日期类型，可以使用日期函数，也可以使用date '2020-01-23 12:23:34'的方式表示为一个日期。还可使用time、timestamp、timestamptz定义对应格式的字符串。

interval ‘<number> <unit>’
interval表示间隔多少，可以不要这个单词

  • number 表示数量
  • unit 表示单位,second、minute、hour、day、week、month、quarter、year都可以。

截取年月日时分秒

使用to_char函数，可进行截取

  • 获取年份,使用YYYY或YY
    select to_char(date ‘1989-12-12’,’YYYY’)
  • 获取月份MM
  • 获取天DD
  • 获取星期,小写表示星期小写，首字母大写为星期首字母大小，全大写为星期字母全大写
    select to_char(date ‘1989-12-13’,’day’)

extract(<unit> from <date>)
该函数表示从date中提取，unit为单位，date为日期时间。但unit中的day表示的是日不是星期
