python及Anaconda安装

2021-10-31

Anaconda安装

进入https://mirrors.tuna.tsinghua.edu.cn/anaconda/archive/ 选择合适的版本，选择Anaconda3开头的是以python3为基础版本的包。2021年4月份的包是3.8版本的python，暂时没有对应稳定tensorflow版本的支持，因此可以向前找一到两个版本。

下载安装时，遇到复选框就勾选，需要添加到环境变量中。

当安装完成后，需要配置国内镜像源，可以选择清华的镜像源配置，在https://mirror.tuna.tsinghua.edu.cn/help/anaconda/中，找到配置文件的代码，并在用户目录下创建.condarc文件，将内容拷贝进去。如果不会创建，则先执行`conda config –add channels https://mirrors.tuna.tsinghua.edu.cn/anaconda/pkgs/free/`自动创建文件，再进行拷贝。

安装tensorflow的时候因为python3.8版本安装不了。name降低python版本，直接执行conda install python=3.7可能不好使，说找不到包。那么我们可以选择创建新的虚拟环境来解决，不使用默认的base环境。
