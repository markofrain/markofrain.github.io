Oracle PL/SQL中执行DML数据操作语言和DDL数据定义语言

2018-06-26

PL/SQL中使用DML和DDL语言

PL/SQL中除了可以执行查询语句外，还可以执行DML数据操纵语言和DDL数据定义语言。

DML数据操作语言语句的使用

示例: 添加一条新纪录，编号为表中最大编号+1

DECLARE
    v_new_id product.productId%TYPE;
BEGIN
    SELECT MAX(productId) INTO v_new_id FROM product;
    v_new_id := v_new_id+1;
    INSERT INTO product VALUES(v_new_id,'MacBook Pro 13.3',12000,1500,NULL);
    COMMIT;
END;

DDL数据定义语言的使用

要想在PL/SQL中使用，需要一条命令来执行，该命令是EXECUTE IMMEDIATE,利用它可以执行动态的SQL语句，也就是不仅可以执行DDL语句，也可以执行DML语句。 示例

DECLARE
    prod_createStr varchar2(200);
BEGIN
    prod_createStr := 'CREATE TABLE TAB_STUDENT
                                (
                                    STUDENTID number(8) primary key,
                                    STUDENTNAME varchar2(20),
                                    AGE nchar(2)
                                )';

    EXECUTE IMMEDIATE prod_createStr;
END;


DECLARE
  v_alter_address varchar2(200);
BEGIN
  v_alter_address := 'ALTER TABLE TAB_STUDENT ADD ADDRESS VARCHAR2(50)';

  EXECUTE IMMEDIATE v_alter_address;
END;

需要注意，将数据定义语言作为字符串，通过EXECUTE IMMEDATE动态执行的时候，最后的分号要去掉，否则错误，这是与一般执行DDL的不同之处。 通常情况下，利用EXECUTE IMMEDATE命令执行DDL语句用在存储过程当中。
