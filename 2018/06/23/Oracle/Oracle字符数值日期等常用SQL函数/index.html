向上向下取整函数
select ceil(-23.4),ceil(23.4),floor(23.5),floor(-23.4) from product where productId=1

由此可见，ceil函数不管n是正数还是负数都往大的走，floor都往小的走。

四舍五入函数

语法：round(n[,i])

select round(3.14),round(3.14159,2),round(3151.5926,-2),round(3141.5926,-5) from product where productId=1
  1. 将n四舍五入，可选参数i表示要保留的位数，不指定则不保留小数。
  2. 参数i要求为整数，若为正数，则保留i为小数，
  3. 负数则虚拟小数点向左i位，虚拟小数点后的一位四舍五入，虚拟小数点至源小数点之间的数变为0，舍去源小数部分。(废话较多理解就好)
  4. 如果i为负数且绝对值大于等于整数部分的长度，则为0
数字截取函数

语法：trunc(n[,i]) 效果与round函数类似，不过小数点后一位并不会四舍五入，而直接截断

select trunc(3.14),trunc(3.14159,3),trunc(3151.5926,-2),trunc(3141.5926,-5) from product where productId=1

字符串截取函数substr

字符串截取函数不光有按字符的，还有字节的等等。 语句结构为 {[substr|substrb|substrc|substr2|substr4]}(char,posistion[,substring_length]) 各参数如下:

  • substr：以字符为单位,中文为一个字符

  • substrb：以字节为单位，中文需要三个字节

  • substrc：以unicode字符为单位

  • substr2：UCS2代码点为单位

  • substr4：以UCS4代码点为点位

  • char：被截取的字符串

  • posistion:要截取的开始位置，初始值为1，第一个字符字节均为1，若为负数则从右往左

  • substring_length：截取的长度，无值截取到末尾。根据截取的类型，如果字节截到汉字时需要3个才能截出来，否则无这个中文。

    select substr(‘abcd我爱你’,2,5),substrb(‘abcd我爱你’,2,5),substrc(‘abcd我爱你’,2,5) from product where productId=1;

select substr('abcd我爱你',-2,5),substrb('abcd我爱你',-3,5),substrc('abcd我爱你',-2,5) from product where productId=2;
字符串搜索函数

instr函数，在指定的字符串中搜索另一个字符串的，不存在返回0，存在返回首字符位置 语句结构 {[substr|substrb|substrc|substr2|substr4]}(string,substring[,posistion[,occurence]]) 语法参数

  • 搜索函数的类型与字符串截取一致

  • string：待搜索字符擦混

  • substring：要搜索的字符串

  • position：搜索开始的位置，默认为1

  • occurence：第几次出现,默认为1

    select instr(‘dabc我爱abc你’,’abc’,1,1),instr(‘dabc我爱abc你’,’abc’,1,2),instr(‘dabc我爱abc你’,’abc’,-1,1) from product where productId=2;

以此为例，最后一个参数设为1(默认为1),则检索第一个匹配的字符串获得位置，在2 检索第二次匹配，字符串的位置是7 那我将position设置为-1，从右往左检索，则第一个匹配到abc的就是位置为7的了。

字母大小写转换

upper,lower,initcap函数

select upper('etvhERGdg'),lower('etvhERGdg'),initcap('this,is,a,test') from product where productId=2;

initcap函数，只要字符串里，几个字符串通过空格，逗号，点 分隔开，则每个字符串的首字母为大写 一般用于标题大写。

字符串替换函数

语法: replace(char,search_string[,replacement_string])

  • char：表示搜索的目标字符串

  • search_string：目标字符串中要搜索的字符串

  • replacement_string：可选，用来替代被搜索的字符串，如无此参数，则删除搜索的字符串

    select replace(‘this is a abcTest’,’abc’),replace(‘this is a abcTest’,’abc’,12) from product where productId=1;

字符串填充函数

rpad(expr1,n[,expr2]) 在字符串expr1的右边用字符串expr2填充，直到字符串长度为n为止，如果expr2不存在则空格填充。 如果n小于字符串expr1的长度，则从左至右截取为n个长度。

select rpad('this is a abcTest',3,'1'),rpad('this is a abcTest',20,'1') from product where productId=1;
字符串首尾空格清除函数

trim,rtrim,ltrim函数 语法: TRIM([LEADING|TRAILING|BOTH] [trim_character FROM] trim_source)

  • LEADING：删除trim_source的前缀字符

  • TRAILING：删除trim_source的后缀字符

  • BOTH：删除trim_source的前缀和后缀字符

  • trim_character:删除的指定字符，默认删除空格,只能是单个字符

  • trim_source：被操作的字符串

    select trim(‘ abcdefga ‘),trim(leading ‘a’ from ‘abcdefga’),trim(trailing from ‘ abcdefga ‘) from product where productId=2;

如果使用LEADING、TRAILING、BOTH，则必须要使用FROM关键字,但是删除指定字符可以是默认空格。 语法: {[RTRIM|LTRIM]|(char,[set]) 右清除和左清除空格函数 1. 默认无set可选参数，则清除左边或右边全部空格 2. 设置set字符为要删除的字符

select ltrim('   my name is abc  '),ltrim('aaaa name is abc','a'),rtrim('   my name is abc  '),rtrim('my name is abc','abc') from product where productId=1;

为日期加上指定月份函数

语法： ADD_MONTHS(date,integer)

  • date：指定的日期

  • integer：要加的月份数，如果为负数，则减月份数

    select add_months(sysdate,2),add_months(‘2018-11-8’,-2) from product where productId=2;

指定日期的下星期几对应的日期

语法: next_day(date,char)

  • date：当前指定日期

  • char：星期几，值可以为，’星期一’ –’星期日’’，以外国周日为每星期的第一天,为空则返回空

    select next_day(sysdate,’’),next_day(sysdate,’星期日’) from product where productId=2;

提取指定日期的特定部分

语法： EXTRACT(YEAR|MONTH|DAY FROM datetime)

select extract(year from sysdate),extract(month from sysdate),extract(day from sysdate) from product where productId=2;

通过指定类型获得当前日期的年、月、日。

示例:

字符转换函数
  1. TO_CHAR(number[,fmt[,nls]])

  • number:数值型数据

  • fmt：转换成字符串的格式

    select to_char(168.568),to_char(168.568,’999.99’),to_char(168.568,’$999.99’) from product where productId=1;

  1. TO_CHAR(date[,fmt[,nls]])

  • nls:使用的语言类型

    select to_char(sysdate),to_char(sysdate,’YYYY-MM-DD’),to_char(sysdate,’HH24:MI:SS’),to_char(sysdate,’J’),
    to_char(sysdate,’MONTH’),to_char(sysdate,’Month’,’NLS_DATE_LANGUAGE=ENGLISH’)
    from product where productId=’1’;

fmt可取值有

  • YYYY-MM-DD：年月日
  • HH24:MI:SS：时分秒
  • YEAR：年份，英文方式,twenty eighteen
  • MONTH：月,6月
  • DAY：星期,星期六
  • J：公元制时间戳
  • YYYY/MM/DD：返回的值为纯数字,年份四位，月日两位
  • ……
日期转换函数

语法: TO_DATE(char[,fmt[,nls]])

  • char：待转换的字符，可以使CHAR、VARCHAR2、NCHAR、NVARCHAR2

  • fmt：转换格式

  • nls：控制格式化时使用语言类型

    SELECT TO_DATE(‘2018-6-23’,’YYYY-MM-DD’),TO_DATE(‘2458293’,’J’) from product where productId=1;

公元制时间戳反过来可以转换成日期

数字转换函数

语法: TO_NUMBER(char[,fmt[,nls]]) 最简单的数字转字符，不需要函数，直接就自动转换

SELECT TO_NUMBER('2345.304'),TO_NUMBER('2345.304','9999.999') FROM product where productId=1;

其他函数

函数名

说明

NVL(expr1,expr2)

如果expr1的值为NULL，则返回expr2，否则返回expr1

NVL2(expr1,expr2,expr3)

如果expr1为NULL，则返回expr3，否则返回expr2

DECODE(value,if1,then1,if2,then2,…,else)

如果value的值为if1，则返回then1；如果值为if2，则返回then2，…，否则返回else

COALESCE(expr)

返回列表中第一个不为NULL的表达式,参数必须是同一类型。

示例:

select nvl(null,'abc'),nvl2('0','abc',123),coalesce(null,5,158) from product where productId=1;

案例较为简单

select decode(to_char(sysdate,'MM'),'01','一月','02','二月','06','六月','其他月份') from product where productId=1;

to_char()函数中，获得值为06，则返回六月。
