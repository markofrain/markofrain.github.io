Oracle数据定义语言，数据操作语言

2018-06-05

数据定义语言数据库表的创建

前面已经有简单的了解数据定义语言，那么直接切入正题。 在创建表之前，先说一下。数据库的问题。 与MySQL数据库不一样，mysql可以创建多个数据库，而Oracle只有一个数据库orcl，它通过用户来操作，每个用户访问到的数据是不一样的。你需要创建用户，创建并分配表空间，赋予权限。并登陆创建的用户来操作。在后面讲，他与sql操作无关，可跳过学习。

创建表

创建语法:

CREATE TABLE table_name
(
column_name datatype [null|not null],
column_name datatype [null|not null],
...
[constraint]
)

语法说明:

  • table_name：数据表名称，在同一数据库中数据表名是不能重复的
  • column_name:列名，或字段名，在一个表中不能重复
  • datatype:数据类型，上节课说的
  • [null | not null]:表明该列是否为空或者不为空
  • [constraint]:设置表中列的约束，主要包括主键约束、外键约束、检查约束等。

示例:

CREATE TABLE product
(
productId number(4) not null,
productName nvarchar2(20),
productPrice number(10,2),
CONSTRAINT pk_product primary key(productId)
)

语法已经说了，约束上章已经讲了，代码就不解释了。

删除表

示例：

DROP TABLE table_name;

删除整张表，删除后将无法使用。

ALTER 语句修改表

如果想更改表的属性或是列，可使用LATER TABLE语句 语法:

ALTER TABLE table_name 
ADD column_name | MODIFY column_name | DROP COLUMN column_name

语法说明:

  • ADD:用于向表中添加列
  • MODIFY:用来修改表中已存在的信息
  • DOP COLUMN:删除表中的列删除列时有时要加上CASCADE CONSTRAINTS,把该列有关的约束一并删掉。

示例:

//假如还是上面创建的product表,要为其添加productWeight重量字段
ALTER TABLE product ADD productWeight number(6,2);

//修改productName的数据类型为varchar2(40),查看数据类型是否修改看下一个代码块
ALTER TABLE product MODIFY productName varchar2(40);

//删除列productWeight字段
ALTER TABLE product DROP COLUMN productWeight;

//更改列名,productName改为productN
ALTER TABLE product RENAME COLUMN productName to productN;

通常删除列字段可能会到你没有权限,无法删除属于sys的表中的列

//查看表数据结构
SELECT table_name, column_name, data_type, char_length
  FROM all_tab_cols
 WHERE table_name = 'PRODUCT' ;

这是表的特性的增删改字段操作，而通过ALTER TABLE添加约束，上个文章中也是说了的。

数据操作语言

INSERT语句

INSERT语句是对表数据的新增操作. 语法:

INSERT INTO table_name(column1，column2，column3...) VALUES(val1,val2,val3...);

如果你要插入整个列的字段,那么不必写列名了

INSERT INTO table_name VALUES(val1,val2,val3...);

当然如果主键是自增的,那就不需要添加主键列了，但是需要为主键列设置序列(后面讲)。 示例:

INSERT INTO product(PRODUCTID,PRODUCTNAME,PRODUCTPRICE,PRODUCTWEIGHT) VALUES(1,'晨光笔',12,20);
INSERT INTO product VALUES(2,'怡宝矿泉水',4,1500);

DELETE语句

DELETE语句用于表数据的删除，根据条件进行删除，一般通过获得主键值删除。 语法:

DELETE FROM table_name WHERE condition

示例:

DELETE FROM product WHERE productId=1;
DELETE FROM product WHERE productName='怡宝矿泉水';

WHERE条件后可以添加任何条件。 如果你想删除整个表格数据，而不删除表结构，可使用如下语句

TRUNCATE TABLE table_name;

区别:

  • DELETE：删除表数据会记录日志，相对比较慢,commit前是可以回滚恢复的
  • TRUNCATE：不记录日志，永久删除，数据不能恢复，速度快，但不会删除表结构
  • DROP TABLE：直接删除整个表，不可恢复

UPDATE 语句

进行数据更改的操作 语法:

UPDATE  table_name SET column1=val1[,column2=val2....] WHERE [condition]

示例:

UPDATE product SET productWeight=550,productName='康师傅小瓶矿泉水' WHERE productId= 2;

一般更改条件根据主键获得，而一般不会更改主键，因为这会导致不好的情况。
