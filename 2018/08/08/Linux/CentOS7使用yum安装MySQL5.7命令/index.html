CentOS7使用yum安装MySQL5.7命令

2018-08-08
Linux
字数统计: 533字   |   阅读时长≈ 2分钟

  1. 下载repo源

    //下载repo源
    wget http://repo.mysql.com/mysql57-community-release-el7-8.noarch.rpm
    //安装源
    rpm -ivh mysql57-community-release-el7-8.noarch.rpm

  2. 安装mysql-server

    //安装mysql-server
    yum install mysql-server

可能报错

解决办法

//删除包依赖
yum remove mysql-libs

然后再次安装

yum install mysql-server

报错:

错误显示：

软件包：mysql-community-server-5.7.20-1.el6.x86_64 (mysql57-community)

需要：libsasl2.so.2()(64bit)

您可以尝试添加 --skip-broken 选项来解决该问题

您可以尝试执行：rpm -Va --nofiles --nodigest

解决办法: 修改vim /etc/yum.repos.d/mysql-community.repo 源文件 [mysql57-community] name=MySQL 5.7 Community Server

baseurl=http://repo.mysql.com/yum/mysql-5.7-community/el/6/$basearch/

baseurl=http://repo.mysql.com/yum/mysql-5.7-community/el/7/$basearch/ enabled=1 gpgcheck=0 gpgkey=file:///etc/pki/rpm-gpg/RPM-GPG-KEY-mysql 最后在重新安装

yum install mysql-server

有y/n提示则写y

yum install mysql-devel

默认配置文件路径： 配置文件：/etc/my.cnf 日志文件：/var/log/var/log/mysqld.log 服务启动脚本：/usr/lib/systemd/system/mysqld.service socket文件：/var/run/mysqld/mysqld.pid 配置mysql文件:

cd /etc/my.cnf 
# 加入配置参数
> character-set-server=utf8

启动服务

service mysqld restart

重置密码

//获得随机密码
[root@localhost ~]# grep "password" /var/log/mysqld.log

mysql -u root -p //进入

//修改密码,包含字母数字特殊字符，至少有一个大写字母
alter user 'root'@'localhost' identified by '你的密码';

//刷新权限
flush privileges
