安装redis

  1. pull最新镜像

  2. 启动容器

    • –restart always 表示启动失败，总是重新尝试
    • -p 表示映射端口
    • -v 表示将/etc/redis映射到本机的/home/centos/spring/redis/config,以及数据data映射到本地
    • –net docker-nat –ip 192.168.0.63 表示为容器指定网卡并赋予ip，此处可以不需要
    • redis 表示是当前镜像名称，没有版本号
    • redis-server /etc/redis/redis.conf 表示以指定配置文件方式启动redis-server
    • –appendonly yes 表示开启持久化

  3. 查看日志，可能会出现没有找到redis.conf的问题

    从官网下载一个标准配置文件,将配置文件放到指定的/home/centos/spring/redis/config下。并修改配置文件:

    • 修改bind为0.0.0.0
    • 修改protected-mode 为 no
    • 修改 daemonize 为 no 关闭守护进程方式启动,改为yes时以外部文件启动方式会失效

  4. 因为配置过restart always,所以配置正确好，自动重启就启动成功了。通过docker inspect container_id查看容器信息。
