Spring JMS ActiveMQ断线重连问题

2020-08-12

连接activemq出现超时，连接失败问题，默认重试3次，服务器在这段时间断掉了。

原因，可能是网络，或者客户端很久没有接受消息断开了。

解决:

断线重连，使用failover机制failover:(tcp://localhost:61616)进行断开自动连接
